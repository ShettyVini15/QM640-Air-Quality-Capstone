# Machine Learning-Based Prediction of Urban Air Quality Using Environmental and Meteorological Data

The project aims to develop and evaluate machine learning regression models for predicting PM2.5 concentration using environmental and meteorological variables from the Beijing Multi-Site Air Quality Dataset. Multiple regression models will be developed and compared to identify the best-performing model for supporting air quality forecasting and environmental decision-making.

**Objectives**
Predict PM2.5 concentration using machine learning regression models.
Compare the performance of multiple regression algorithms.
Identify the most influential environmental and meteorological variables using feature importance analysis.
Improve model interpretability using SHAP visualizations, where appropriate

## Overview

This repository contains the QM640 Data Analytics Capstone project titled **Machine Learning-Based Prediction of Urban Air Quality Using Environmental and Meteorological Data**.

**Source:** https://archive.ics.uci.edu/dataset/501/beijing+multi+site+air+quality+data
