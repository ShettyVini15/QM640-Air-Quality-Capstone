# Machine Learning-Based Prediction of Urban Air Quality Using Environmental and Meteorological Data

## Overview

This repository contains the QM640 Data Analytics Capstone project titled **Machine Learning-Based Prediction of Urban Air Quality Using Environmental and Meteorological Data**.

The project aims to develop and evaluate machine learning regression models for predicting PM2.5 concentration using environmental and meteorological variables from the Beijing Multi-Site Air Quality Dataset.

## Dataset

**Source:**
https://archive.ics.uci.edu/dataset/501/beijing+multi+site+air+quality+data

The dataset is publicly available through the UCI Machine Learning Repository.
