# Data

This project uses the **Beijing Multi-Site Air Quality Dataset** from the UCI Machine Learning Repository.

## Dataset Source

https://archive.ics.uci.edu/dataset/501/beijing+multi+site+air+quality+data

## Instructions

The dataset is publicly available and is **not included** in this repository due to its size and licensing terms.

To reproduce this study:

1. Download the Beijing Multi-Site Air Quality Dataset from the UCI Machine Learning Repository.
2. Extract the downloaded files.
3. Copy all dataset CSV files into the **data/** folder.
4. Run the notebook **Dataset_Exploration.ipynb** located in the **notebooks/** folder.
